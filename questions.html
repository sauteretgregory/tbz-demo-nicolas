Méthode TalentBusterZ — Static UX V0.6

La version statique n’a pas de base de données serveur. Elle utilise un fichier RAW JSON comme mémoire locale portable.

Règle utilisateur

TalentBusterZ propose. L’utilisateur décide : utiliser dans ce CV, conserver dans le RAW, modifier/préciser ou supprimer du RAW.

RAW JSON : fichier provisoire jouant le rôle de base de données locale et de mémoire IA. Il doit être conservé et fourni à la prochaine analyse avec le CV source.
